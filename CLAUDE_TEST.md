# Claude Code

Claude Code is Anthropic's official command-line interface (CLI) tool that brings Claude directly into your terminal. It acts as an interactive AI agent that assists with software engineering tasks such as writing code, debugging, refactoring, navigating codebases, running commands, and managing git workflows. Claude Code understands your project context by reading files, searching code, and executing commands, enabling it to provide relevant, grounded assistance without leaving your development environment.
